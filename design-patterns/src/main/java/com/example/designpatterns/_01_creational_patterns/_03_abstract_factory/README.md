# 추상 팩토리 패턴
>서로 관련있는 여러 객체를 만들어주는 인터페이스
 
- 구상 클래스에 의존하지 않고도 서로 연관되거나 의존적인 객체로 이루어진 제품군을 생산하는 인터페이스를 제공한다. 구상 클래스는 서브클래스에서 만든다.
- 구체적으로 어떤 클래스의 인스턴스(concrete product)를 사용하는지 감출 수 있다.
